# keyp - Onramper Widget

### How to run locally

`git clone https://github.com/geovgy/keyp-onramper`


`cd keyp-onramper/frontend`


`yarn`


`cp .env.example .env`


set the REACT_APP_ONRAMPER_API_KEY value in the `.env` file


`yarn start`


Open the app in your browser at http://localhost:3000
